# Cách hoạt động của Internet trong 5 phút

Internet là mạng lưới toàn cầu kết nối hàng tỷ thiết bị. Khi bạn nhập một địa chỉ web, trình duyệt sẽ:

1. Tra cứu DNS để lấy địa chỉ IP của server.
2. Gửi yêu cầu HTTP tới server đó.
3. Server xử lý và trả về dữ liệu (HTML, CSS, JS...).
4. Trình duyệt hiển thị nội dung cho bạn.

Các giao thức như TCP/IP đảm bảo dữ liệu truyền đi an toàn, đúng thứ tự. Nhờ đó, bạn có thể truy cập web, gửi email, xem video... chỉ trong vài giây!
