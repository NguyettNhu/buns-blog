# Giải thích Clean Code với ví dụ thực tế

Clean Code là code dễ đọc, dễ hiểu, dễ bảo trì. Một số nguyên tắc:

- Đặt tên biến/hàm rõ nghĩa
- Hàm ngắn, làm 1 việc
- Tránh lặp code
- Comment hợp lý

Ví dụ:

```js
// Code chưa clean
function fn(a, b) { return a + b }

// Clean code
function sum(a, b) { return a + b }
```

Code sạch giúp teamwork hiệu quả, giảm bug và dễ mở rộng.
