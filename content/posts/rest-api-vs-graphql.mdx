# REST API vs GraphQL: Nên chọn cái nào?

REST API là chuẩn truyền thống, mỗi endpoint đại diện cho một tài nguyên. GraphQL cho phép client truy vấn dữ liệu linh hoạt chỉ với một endpoint.

- **REST**: Đơn giản, dễ cache, tốt cho API public.
- **GraphQL**: Truy vấn động, giảm over-fetching, phù hợp app phức tạp.

Chọn REST nếu API đơn giản, cần ổn định. Chọn GraphQL nếu muốn tối ưu truy vấn, phát triển nhanh, hoặc app mobile/web hiện đại.
